# Session Transcript: Stage 009 — Gini Inequality Analysis

## Session Overview

The session implemented GP-based Sobol sensitivity analyses for the `gini_k_final`
estimand across both the baseline and bivariate σ models, tested the hypothesis that
higher baseline escalation raises equilibrium centrality inequality, and extended
`docs/final-report.md` with a closing section on the Gini findings.

## Design Decisions Made

### Decision 1: Absolute `gini_k_final` level as primary estimand
**Chosen:** `gini_k_final` is analysed as an absolute level across the full
parameter space, not as a paired difference between mu0 conditions.
**Rationale:** A paired difference (Gini_hi − Gini_lo) conflates the question of
whether escalation raises inequality with the question of how structural parameters
govern the baseline level; the absolute level is the interpretable quantity for
understanding which configurations produce concentrated versus distributed degree
centrality.
**Tradeoffs:** Requires treating the mu0 comparison as a separate hypothesis test
rather than folding it into a single derived estimand.
**Proposed by:** agent

### Decision 2: Matched parameter sets for direct Sobol comparability
**Chosen:** Baseline Gini Sobol uses the same four parameters (alpha, gamma, lambda,
eta_obs) as the Stage 003 Ψ and ε–degree analyses; bivariate Gini Sobol uses the
same six parameters as Stage 007.
**Rationale:** Using the same reduced parameter sets enables direct rank-order
comparison across estimands (Ψ / ε–degree / Gini in the baseline; Ψ / psi_sigma /
Gini in the bivariate). Introducing a different parameter set would prevent the
key cross-estimand comparison.
**Tradeoffs:** Cannot detect whether parameters excluded from the reduced set
contribute to Gini variance.
**Proposed by:** agent

### Decision 3: Dissipative inequality as secondary estimand
**Chosen:** `gini_dissipative = gini_peak − gini_k_final` is computed alongside
`gini_k_final` and characterised structurally.
**Rationale:** The gap between peak transient Gini and equilibrium Gini reveals
how much of escalation's concentration effect the cooperative reconstruction
mechanism subsequently reverses — a quantity that connects directly to the
cooperative-reconstruction account developed in the final report.
**Tradeoffs:** Adds four additional GPs to the fitting pipeline; secondary estimand
is reported at shorter length than the primary.
**Proposed by:** agent

### Decision 4: "Remaining questions" section retained for two non-Gini open items
**Chosen:** The existing "Remaining questions" section in `docs/final-report.md`
is updated to remove the resolved Gini item but retain the two still-open questions
(Ψ=1 threshold under σ; bivariate centrality correlation under σ).
**Rationale:** The task specified removal once all three deferred items are
addressed; only one (Gini inequality) was addressed in this stage. The remaining
two are substantive open questions and should not be silently dropped.
**Tradeoffs:** The section is no longer a clean "all open questions resolved"
outcome; the two retained questions are explicitly deferred.
**Proposed by:** mpadge

## Tradeoffs Considered

- **Paired Gini difference vs. absolute level**: the paired approach would have
  mirrored the Ψ estimand design and made the mu0 effect directly comparable to Ψ
  in scale; the absolute-level approach won because the governing-parameter question
  (which structural variables control inequality?) is the primary question, and the
  paired quantity cannot answer it without the baseline level.

## Open Questions

- Whether the Ψ = 1 amplification threshold shifts under σ heterogeneity.
- Whether the Stage 003 ε–degree dissociation result holds in the bivariate model.
